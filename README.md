## üöÄ Getting Started

### Prerequisites

You will need [Node.js](https://nodejs.org/) installed to run the local development server.

### Local Development

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd art-portfolio-spa
   ```

2. **Run the local server**:
   You can use `npx serve` to launch the application locally:
   ```bash
   npx serve .
   ```
   *Alternatively, specify a port:*
   ```bash
   npx serve -l 8000 .
   ```

3. **Open in Browser**:
   Navigate to `http://localhost:3000` (or the port specified).
