# Art Portfolio Single Page Application

A modern, visually stunning single-page web application showcased as a creative portfolio of artworks from middle and high school years (2022-2026). This project features a curated collection of paintings, drawings, digital art, printmaking, and architectural designs.

Hosted by github page: https://penguin12b5.github.io/art-portfolio-spa

## ‚ú® Features

- **Dynamic Gallery**: A responsive grid layout with smooth reveal animations using the Intersection Observer API.
- **Smart Filtering**: Categorize and browse artworks by medium (Paintings, Drawings, Printmaking, Digital Art, etc.).
- **Immersive Lightbox**: View artworks in high resolution with detailed metadata, including medium, dimensions, and year of creation.
- **Modern Aesthetics**: Premium design featuring glassmorphism, smooth transitions, and a curated color palette.
- **Contact Integration**: A functional contact form for inquiries and print requests, integrated via Formspree.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## üöÄ Getting Started

### Prerequisites

You will need [Node.js](https://nodejs.org/) installed to run the local development server.

### Local Development

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd art-portfolio-spa
   ```

2. **Run the local server**:
   You can use `npx serve` to launch the application locally:
   ```bash
   npx serve .
   ```
   *Alternatively, specify a port:*
   ```bash
   npx serve -l 8000 .
   ```

3. **Open in Browser**:
   Navigate to `http://localhost:8000` (or the port specified).
